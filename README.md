# **🚀 这个项目是干啥的？**

一句话：给那些满嘴“科技”、技术却还停留在清朝的互联网编辑器“立个碑”。

---

2026 年了，WebP 早就是互联网的基建了。

辛苦写好的文章，往某些号称“精品”的平台一发，反手就给一个“格式不支持”。

这种为了适配平台而强迫用户改格式的行为，说白了就是技术上的傲慢与无能。

这个项目由 GokuScraper (悟空爬虫) 团队发起，旨在建立一个“编辑器兼容性实时黑榜”。

📊 收录标准：

不支持 WebP 上传： 报错或直接拦截。

不支持 WebP 处理： 上传后图裂了或者无法预览。
