🚀 这个项目是干啥的？
一句话：给那些满嘴“生产力”、技术却还停留在清朝的互联网编辑器“立个碑”。

2026 年了，WebP 早就是互联网的基建了。可咱们辛苦写好的文章，往某些号称“精品”的平台一发，反手就给一个“格式不支持”。这种为了适配平台而强迫用户降级资产的行为，说白了就是技术上的傲慢与无能。

这个项目由 GokuScraper (悟空爬虫) 团队发起，旨在建立一个**“编辑器兼容性实时黑榜”**。

🔨 咱们的打击目标：
伪装派： 口号喊得响，连 WebP 上传都跑不通。

装死派： Issue 提了几年，死活不修，就让用户手动转 JPG。

草台班子： 整个公司可能就一个运维，根本没技术去改老旧的图片处理逻辑。

📊 收录标准：
不支持 WebP 上传： 报错或直接拦截。

不支持 WebP 处理： 上传后图裂了或者无法预览。

技术傲慢： 客服回复“请您自己转成 JPG 呢”。
